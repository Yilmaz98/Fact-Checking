Documentation:


To get the Tweets on Twitter:

python Extractor.py --querysearch "Enter your query" --since YYYY-MM-DD --maxtweets "Count" --output "Output file name"


Example:

Justin Trudeau,the Canadian Prime Minister has come to India. (Date: 23rd February 2018)

python Extractor.py --querysearch "Justin Trudeau India" --since 2018-02-23 --maxtweets 500 --output trudeau.csv
